# Agent Operating Contract

## Foundation-first rule
Always build or verify persistence and hydration before adding media-generation features.

## No-drift rule
Do not introduce parallel reasoning systems for center modes. All center modes must derive from one shared structured session-analysis state.

## Branch behavior rule
- Branches are suggested by analysis.
- Branches open only when user chooses them.
- Branch list lives in the far-left rail.

## Disclosure rule
The optional thread-area disclosure label must be exactly:
- What this is based on

## Validation rule
For workspace-focused tasks:
- run lint
- run typecheck
- run build
- report blockers instead of guessing when contract docs are missing or ambiguous.
